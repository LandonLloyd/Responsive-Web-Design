Beginner Bike

A beginner bike is usually a hardtail with a suspension fork. One of these will run about $300.

Hardtail

A hardtail is a bike with only front wheel suspension. Hardtails if used correctly used can be lots of fun. The range from $400 to about $3,000.

Full-Suspension

Like the name implies this bike has suspension on both wheels. These bikes can range from about $400 to about $12,000.
